How to use in a script

Install claude-agent-sdk in the script's Environment, then:

from pyrunner_ai import ask_claude

answer = ask_claude("Search the web for today's AI news and summarize it")
print(answer)

Scripts always use the active provider. Web search/fetch tools are available on Anthropic only; other providers still support your own tools and plain prompts.

For an Anthropic subscription token, run claude setup-token on your own machine (logged into Claude) and paste the token it prints (starts with sk-ant-oat01-) — not the browser authorization code.
