Using databases in scripts

Grant a database to a script on the database's page, then connect with the pyrunner_db helper — a real PostgreSQL connection, scoped to this database only. Requires psycopg[binary] in the script's environment.

import pyrunner_db with pyrunner_db.connect("{{ databases.0.name|default:'my_database' }}") as conn: conn.execute("CREATE TABLE IF NOT EXISTS items (id serial PRIMARY KEY, data jsonb)") conn.execute("INSERT INTO items (data) VALUES (%s)", ('{"hello": "world"}',)) rows = conn.execute("SELECT * FROM items").fetchall()
